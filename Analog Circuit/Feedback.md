# General Feedback System

# Feedback Topology

| Topology | Input Impedance | Output Impedance |
|----------|-----------------|------------------|
| Voltage-Voltage | $R_{if} = R_i(1+A_v\beta_v)$ | $R_{of} = \dfrac{R_o}{1+A_v\beta_v}$ |
| Voltage-Current | $R_{if} = R_i(1+G_m\beta_r)$ | $R_{of} = R_o(1+G_m\beta_r)$ |
| Current-Voltage | $R_{if} = \dfrac{R_i}{1+R_m\beta_g}$ | $R_{of} = \dfrac{R_o}{1+R_m\beta_g}$ |
| Current-Current | $R_{if} = \dfrac{R_i}{1+A_i\beta_i}$ | $R_{of} = R_o(1+A_i\beta_i)$ |
