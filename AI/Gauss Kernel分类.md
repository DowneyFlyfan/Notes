# Gauss Kernel分类
1.  Isotropic Gauss Kernel  
      
    \\[k(x,y) = \frac{ 1 }{ 2 \pi \sigma^2 } e^{ -\dfrac{ x^2 + y^2 }{ 2 \sigma^2 }}]  
    
2.  Anisotropic Gauss Kernel  
    \\[k(x,y) = \frac{ 1 }{ 2 \pi \sigma_x \sigma_y } e^{ - \dfrac{ x^2 }{ 2 \sigma_x^2 } - \dfrac{ y^2 }{ 2 \sigma_y^2 } }]
