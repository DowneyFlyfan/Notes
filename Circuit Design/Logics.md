# Logic Operators & Logic Gates
1.  **Logic Operators**  
      
    AND: (F = x \cdot y$, (F = xy$  
    OR: (F = x + y$  
    NOT: (F = x'$, (F = \overline x$  
    XOR: (F = x \oplus y = x'y + xy'$  
    NXOR: (F = x \ominus y = xy + x'y'$
